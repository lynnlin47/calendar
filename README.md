# ปฏิทิน
แค่นั้นแหละ ไม่ได้มีอะไรน่าหวือหวาอะไร

รัน xampp จับไฟล์ index.php เข้าไปที่ C:\xampp\htdocs แค่นั้น ก็พร้อมใช้แล้ว
หรือ ถ้าต้องการทดสอบก็แค่ สร้างโฟลเดอร์ใน htdoces เช่น test และจับไฟล์นี้เข้าไปวางแค่นั้นก็พอ

เวลาเข้าเว็บก็เขียน localhost หรือ localhost/test/ หรือ 127.0.0.1 หรือ 127.0.0.1/test/

ปล. โปรเจคนี้ทำใช้เองแค่นั้นแต่อยากเอามาแบ่งปั่นค่ะ และ อาจจะมีอัพเดทอะไรขึ้นมาเรื่อยๆในอนาคต หากมี จะมีไฟล์ที่ชื่อ update.md ขึ้นมาว่ามีอัพเดทอะไรบ้าง
